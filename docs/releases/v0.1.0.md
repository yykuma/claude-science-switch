# Claude Science Switch v0.1.0

Claude Science Switch v0.1.0 是首个公开版本：通过本地 loopback 代理接管 Claude Science 请求，把推理切到你自己的模型 API，并在 Claude Science 模型选择器里显示真实上游模型名。

## 概览

这一版提供 macOS Apple Silicon 桌面包和 Linux x64 CLI 包，支持多 provider 配置、协议转换、模型目录生成和隔离 profile smoke。默认路径只监听本地 loopback，不读取或修改真实 Claude Science 登录态。

## 重点内容

- **本地 Claude Science 代理**：用本地代理接管 Claude / Anthropic 兼容请求，并转发到你配置的 provider。
- **多 provider 切换**：支持 cc-switch 风格 provider 配置，可列出、切换和临时指定 provider。
- **多协议支持**：支持 `anthropic`、`openai_chat`、`openai_responses`、`gemini_native`。
- **真实模型名显示**：Claude Science 模型选择器显示真实 upstream model，避免多个模型都显示为 `Default`。
- **模型目录去重**：按真实 upstream model 去重，每个真实模型对应一个可见条目。
- **隔离 profile smoke**：测试运行使用临时 profile，不读取或修改真实 Claude Science 登录态。

## 下载与安装

访问 [Releases](https://github.com/yykuma/claude-science-switch/releases/tag/v0.1.0) 下载对应版本。

### macOS

| 系统 | 架构 | 文件 |
| --- | --- | --- |
| macOS 12+ | Apple Silicon | `Claude.Science.Switch_0.1.0_aarch64.dmg` |

下载 DMG 后，把 `Claude Science Switch.app` 拖入「应用程序」。首次打开如被系统拦截，右键应用选择「打开」。

### Linux

| 系统 | 架构 | 文件 |
| --- | --- | --- |
| Linux | x86_64 | `claude-science-switch_0.1.0_linux-x64.tar.gz` |

```bash
tar -xzf claude-science-switch_0.1.0_linux-x64.tar.gz
cd linux-x64
./bin/claude-science-switch doctor --config examples/cliproxy-gpt55.json
./bin/claude-science-switch serve --config examples/cliproxy-gpt55.json
```

控制台默认地址：

```text
http://127.0.0.1:17777/
```

## 风险提示

Claude Science Switch 只负责本地路由和协议转换。使用第三方 provider 时，请确认你了解对应 provider 的计费、数据保留和服务条款。
